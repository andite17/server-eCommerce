## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  # Using npm
  npm install

  # Using yarn
  yarn install
```

Start the server

```bash
  # Using npm
  npm run start

  # Using yarn
  yarn start
```
